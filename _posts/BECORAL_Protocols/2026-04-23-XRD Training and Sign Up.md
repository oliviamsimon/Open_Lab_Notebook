# XRD Training and Sign Up

*X-ray diffraction is a non-distructive technique to determine the mineralogy of a sample. By definition ([of Bragg’s Law](https://en.wikipedia.org/wiki/Bragg%27s_law)), it works on crystalline samples. Hence, it is minimally informative if you’re interested in amorphous phases. Relative phase abundance of multi-phase crystalline samples can be determined using the [Relative Intensity Ratio](https://journals.iucr.org/paper?s0021889874010375) against consistent reference patterns.*

## XRD Use (D2)

### SAMPLE HOLDER

1. If needed for your sample, you can wear gloves and clean the sample holder and any other tools with ethanol or water and then wipe with Kimwipes. Gloves and cleaning materials should be provided by the user.

2. Zero background diffraction plate:

Usually lives on the same benchtop as the instrument or the table immediately to the right.

Is opaque circle with a silver circular plate in the middle (~5 cm diameter total)

Is from the D8 diffractometer but can be used in the D2.

Diffracts at ~1o 2ϑ from 0

### Small sample sizes

1. If small samples have small particle sizes, you can put a very small smear of petroleum jelly (bottle found on the table to the right of the instrument) on the sample holder and then wipe most of it off with a Kimwip before putting the sample on top. This prevents fine-grained powder being blown off.

2. Because of the petroleum jelly, the sample is minimally recoverable. It will also contaminate the sample and should not be used if post-XRD elemental or isotopic analysis is desired on the same sample.

3. The petroleum jelly can be cleaned off the sample holder with a squirt of acetone (bottle found on the table to the right of the instrument) and then wiped off with a Kimwipe. Make sure to leave the sample holder clean after use.

4. Pat down any lumps on the line of sample before analysis to increase crystal alignment.

### Installation in instrument

1. Pull the latch at the base of the front door to open it. Door swings upward.

2. Metal box with yellow hazard triangle (for heat) should feel warm as it is heated to 50oC.

3. The sample should be aligned with the shadow on the sample holder; this strip of shadow is where the x-rays hit.

4. Wear your ring dosimeter when installing and removing samples on the instrument.

## INSTRUMENT SET UP

### Slits

1. Source slits to the left of the sample holder can be installed for either 0.2 or 0.6 mm source restrictions. A larger source slit allows more incident x-rays hit the sample, increasing signal but also allows more noise. A smaller slit reduces noise but also reduces signal.

2. Install the source filter with the filter closer to the source on the left; numbers will be facing outward.

3. The detector to the right of the sample holder does not have a slit. This allows all diffracted x-rays to be detectable, but also allows increased noise.

4. Start with the 0.6 mm slit. After an initial run, switch to the 0.2 mm slit if the background is too high.

5. Pull the metal handle with the black knob forward (it is located at the bottom of the metal box with the yellow hazard triangle) and upward until it stops to get the sample holder into place.

6. Pull the outer door downward to close. X-rays will not be generated if the outer door is open and do not come on automatically when the door is closed.

7. The screen for the D2 appears to be a touchscreen but it is NOT. Use the keyboard and mouse.

8. Ensure voltage and current are set to 30 kV and 10 mA, respectively (the numbers to the right under X-Ray Generator. To change the settings, enter new values and click Set; you shouldn’t need to do this.

9. Click the On box next to the right of X-Ray to turn on the x-rays. The actual voltage and current numbers will increase to the set values.

## INITIAL/FAST SCAN

### Settings

1. Set Increment (2ϑ) to 0.1o.

2. Set Start and Stop angles.

3. Set Time/Step (s) to 0.5.

4. The Total Time (s) will autopopulate.

5. Set PSD to 4o. This setting can’t go higher than 5.

6. Parking Position should be set to 60o 2ϑ.

7, Voltage should be set to 30 V.

8. Current should be set to 10 mA.

9. Click Start.

10. You can tell that the run is complete when Unit Actual 2ϑ is back to 60o.

11. The run should take ~5 minutes.

12. Click the Off box next to the right of X-Ray to turn off the x-rays. The door should NOT open if the x-rays are on (for safety).

13. Remove the zero background diffraction plate. If desired, transfer the sample back to its tube using weigh paper; otherwise, dispose of non-hazardous samples in the trash.

14. Clean the zero background diffraction plate with 70% EtOH and a KimWipe between samples and at the end of your session.

## “INDUSTRY STANDARD” SCAN

1. Set Increment to 0.02o 2ϑ.

2. If you want a faster run with this increment, decrease the time per step

3. All other settings can remain the same unless you want to change the Start and Stop angles.

4. Good 2ϑ range is 10-75o

5. 10-20o is for air scatter and background diffraction

6. Low angle diffraction also corresponds to larger crystal sizes which is increasingly possible with included organic matter (>5%)

7. This run could take 15-60 minutes depending on settings.

8. Signal to noise should be MUCH better.

9. Primary peaks should be at least 1000 detections.

## RETRIEVING DATA

1. There is no internet connection to the D2. Data must be transferred by USB: bring your own.

2. Two file types are produced:

.raw is essentially just an xy file. It is the smaller of the two files.

If you’d like the file as an xy file for graphing, click Save -> Save Results. Choose .txt as the file type.

.brml has additional formatting data. It is the larger of the two files.

If you’d like the file as an xy file for graphing, remove the header text and then click Save -> Save Results. Choose .txt as the file type.

Both files contain all necessary data for post-processing.

Both file types can be read by X’Pert HighScore Plus software.

3. After retrieving the desired file type(s), do NOT eject the USB. There is a USB connection for the instrument itself. JUST REMOVE THE USB FROM ITS PORT.

## X’PERT HIGHSCORE PLUS SOFTWARE

1. This software is on the Desktop of the X’Pert Powder Diffractometer computer.

2. Use your NetID for login.

3. Open the HighScore Plus software.

4. Drag the relevant sample file (.raw, .xy, or .brml) from the USB into the HighScore Plus software.

5. The first time you do this, you must tell the software to recognize these file types as they are non-standard Phillips instruments.

6. Customize -> Program Settings -> Automatic Processing -> Add Extensions

7. Type in the extension (e.g., “raw”) and click OK.

8. Identify the background.

Treatment -> Determine Background

This puts a green background line

Click Zoom Intensity button on the bottom line, drag the box to zoom in

Raise the Bending Factor or lower the Granularity

Usually works to set the bending factor to 0 and granularity to 20 to remove air scatter

Add a minimum intensity

Scan List -> Shifts -> Add (cts) if minimum intensity is negative

Click Accept

Treatment -> Clip Range to get rid of small angle region

9. Find the diffractogram peaks


Treatment -> Search Peaks

Minimum Significance = 1

Minimum Tip Width (2ϑ) = 0.1, change it until it finds multiple peaks in a small range (zoomed in)

Click Search Peaks to test new settings

Click Accept when all desired peaks are found

10. Peak List (window)

Zoom in

Delete not-real peak rows (can control-Z if a real peak is accidentally deleted)

Can drag columns to the left to see their most important information

Can drag the spectrum cursor to check peaks at higher 2ϑ

11. Clean up profile

Treatment -> Profile Fit -> Fit Profile

Peak List -> Check FWHM (full-width at half-maximum) for any peaks that are much larger than the others

Can delete 𝛼2 hatches by right-clicking -> Strip K alpha 2

12. Determine mineralogy(ies)

-> Search + Match -> Execute Search + Match

Parameters -> Peak + Profile Data -> Multi phase

Restrictions: start with none or set elements of interest as known (present or absent) or possible (present)

Click Search

13. Search Hits window

Anything in yellow is good, green is very good

Higher score is better. Can click Score to rearrange order

Right-click to Customize Candidate List -> drag RIR into Search Hits or double-click it to add

RIR (reference intensity ratio) is how much better that reference code scatters than corundum does

If missing, add Mineral Name as well

Can click Compound Name or Chemical Formula to rearrange order to search for specific compounds or elements

Choose the Ref. Code that has the best RIR for its score of the desired mineral type

Click Accept to choose a Ref. Code

The remainder of the search hits will re-sort to get rid of other equivalent matches (mineralogies)

If only low-score matches remain (25 and below), then the sample is likely single phase

Repeat the review of search hits for likely mineralogies if relative quantification (i.e., percent composition) is desired

If the secondary phase is ~0.25% of the whole composition, you may only find low angle peaks. Usually in corals, you will have a primary peak (20-35o 2ϑ) in addition to any low angle peaks.

14. Reports

Report -> Create Report

Emge version of html gives a pie chart of the phases

Report auto-saves in the same place as the .hpf files

Save as HPF file type

HighScore Plus file type

15. Sign off of the X’Pert computer when you’re done
